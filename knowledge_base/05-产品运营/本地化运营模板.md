## 5. 文案检查规则

| 检查项 | 要求 |
| --- | --- |
| 语义 | 保留用户收益，不逐字翻译中文 |
| 长度 | 标题、按钮、弹窗必须适配目标语言长度 |
| 时区 | 活动截止、Push 发送必须使用当地时间 |
| 货币 | 价格、折扣、订阅周期使用当地习惯 |
| 文化 | 避免不适合当地的节日、颜色、符号和人物设定 |
| 合规 | 订阅、试用、退款、广告奖励描述不能误导 |
| 语气 | 美国市场可直接，欧洲市场注意隐私和合规表达，日韩市场注意礼貌和上下文 |

## 8. 决策规则

| 信号 | 判断 | 动作 |
| --- | --- | --- |
| 某语言点击低但关闭不高 | 文案吸引力不足 | 改标题/CTA |
| 点击高但转化低 | 承诺和落地页不一致 | 改落地页或权益解释 |
| 某国家退款/取消高 | 价格或订阅理解问题 | 重写 paywall 和客服解释 |
| 某国家广告投诉高 | 广告格式/频次/素材不适配 | 降频、排查广告源、更新说明 |
| 本地用户集中指出翻译问题 | 机器翻译或语境错误 | 人工重写并复查所有入口 |
