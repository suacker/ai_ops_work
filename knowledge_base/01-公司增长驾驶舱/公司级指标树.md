# 公司级指标树

## 北极星指标

公司级经营不使用单一北极星指标替代全部判断。顶层看“可持续利润增长”，拆成 4 条主线：

| 主线 | 核心指标 | 经营含义 | 常见 owner |
| --- | --- | --- | --- |
| 增长规模 | Spend、Installs、DAU、新增用户 | 是否有足够增长入口 | UA / 增长 |
| 回收质量 | ROI0、ROI7、ROI30、Profit、CPI | 钱是否投得有效 | UA / BI |
| 变现效率 | IAA Revenue、IAP Revenue、ARPDAU、eCPM、IPU、Paywall CVR | 用户是否产生收入 | 变现 / 产品 |
| 产品健康 | D1/D7 Retention、核心行为率、崩溃率、退款率、负反馈 | 收入是否可持续 | 产品 / 运营 / 研发 |

## 指标树

| 层级 | 指标 | 下钻维度 | 常见动作 |
| --- | --- | --- | --- |
| 公司 | Revenue | 产品、国家、平台、变现类型 | 调整资源优先级 |
| 公司 | Profit | 产品、渠道、国家、Campaign | 调整预算和变现策略 |
| 买量 | Spend | 渠道、账号、国家、Campaign、adgroup | 加预算、降预算、迁移预算 |
| 买量 | CPI / CPA | 渠道、国家、素材、版位 | 优化素材、调整定向、控制测试预算 |
| 买量 | ROI0 / ROI7 / ROI30 | cohort、国家、渠道、Campaign | 放量、观察、暂停候选 |
| 广告变现 | Ads Revenue | 产品、国家、广告格式、广告场景、mediation source | 修复配置、调整频控、优化 waterfall/bidding |
| 广告变现 | eCPM / IPU / Fill / Show Rate | 广告格式、场景、国家、版本 | 定位流量质量或展示链路问题 |
| 订阅 | Trial Start / Trial Conversion / Renewal / Refund | 国家、价格、入口、版本、来源 | Paywall、价格、权益、退款治理 |
| 产品 | D1 / D7 Retention | 国家、版本、渠道、用户类型 | 优化激活、核心路径、Push、版本 |

## 状态判定建议

| 状态 | 判定条件 | 可做动作 |
| --- | --- | --- |
| 健康 | 核心指标达标，数据完整，无重大异常 | 放量、扩素材、扩大实验 |
| 观察 | 单项指标偏离或样本不足 | 小预算验证、补数、下钻定位 |
| 高风险 | 两项以上核心指标偏离，或影响利润/收入/稳定性 | 降风险、暂停候选、修复配置 |
| 数据缺失 | 关键数据无法支撑判断 | 数据修复、归因对账、权限处理 |

## 使用边界

- 公司级指标用于排序和资源决策，不替代专题分析。
- ROI 预测值必须标注 forecast，不得与真实已发生 ROI 混用。
- IAA 和 IAP 产品不要只用同一指标排名，必须结合商业模式。
- 高收入但高退款、高投诉或低留存的产品不能直接判定健康。
