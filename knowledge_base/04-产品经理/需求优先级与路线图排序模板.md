## 评分模型

| 维度 | 评分 | 判断说明 |
| --- | ---: | --- |
| 用户价值 | 1-5 | 是否解决高频、高痛、强意图问题 |
| 商业价值 | 1-5 | 是否影响收入、留存、买量回收或成本 |
| 证据强度 | 1-5 | 是否有真实数据、用户反馈、实验或竞品事实 |
| 影响范围 | 1-5 | 覆盖国家、平台、人群、收入占比或 DAU |
| 战略匹配 | 1-5 | 是否服务当前季度主要增长约束 |
| 实现成本 | 1-5 | 5 表示成本高，需要在总分中扣减 |
| 风险 | 1-5 | 5 表示风险高，需要在总分中扣减 |

建议总分：

```text
总分 = 用户价值 + 商业价值 + 证据强度 + 影响范围 + 战略匹配 - 实现成本 - 风险
```

## 优先级定义

| 优先级 | 定义 | 进入条件 | 输出要求 |
| --- | --- | --- | --- |
| P0 | 当前阶段必须做，直接影响核心指标或重大风险 | 高价值、高证据、时间敏感或阻断增长 | PRD、埋点、验收、上线监控齐全 |
| P1 | 应该做，能改善关键指标但不阻断当前版本 | 价值明确，证据中高，成本可控 | 可进入近期版本或实验 |
| P2 | 可做，作为补充优化或探索 | 价值存在但证据弱、影响小或成本高 | 先做 discovery 或轻量实验 |
| P3 | 暂不做 | 价值弱、证据不足、机会成本高 | 写清不做原因和重新评估条件 |

## 证据清单

| 证据类型 | 可接受内容 | 不足时的处理 |
| --- | --- | --- |
| 定量数据 | Snowball 漏斗、留存、ROI、广告变现、订阅/IAP 指标 | 标记低置信度，不给强动作 |
| 用户反馈 | 客服、商店评论、访谈、可复现录屏 | 补充样本量和代表性说明 |
| 竞品事实 | 可复现路径、截图、价格、广告场景 | 不能直接等同于适合我们 |
| 技术约束 | 研发评估、依赖系统、性能风险 | 拆阶段或降级方案 |
| 商业约束 | 买量计划、广告源、订阅价格、合规要求 | 明确协同 Owner |

## 重新评估触发条件

- 关键指标连续多个完整自然日异常。
- 买量预算、国家结构或渠道结构发生明显变化。
- 订阅退款、广告投诉、崩溃或 ANR 触发护栏。
- 竞品发布强相关能力并影响商店排名或素材卖点。
- 研发成本因技术方案变化明显降低或升高。
