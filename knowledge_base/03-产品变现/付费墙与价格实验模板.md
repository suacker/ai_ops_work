# Paywall 与价格实验模板

## 适用场景

用于设计、执行和复盘 Paywall、订阅价格、试用策略、默认商品、权益表达、折扣、入口触发和本地化价格实验。实验目标必须同时覆盖短期收入和长期质量，不能只用首日收入判断成功。

## 实验假设

```text
如果在 {scene/user_segment/country} 将 {paywall/price/trial/offer} 从 {control} 改为 {treatment}，
那么 {primary_metric} 会提升，
原因是 {user_insight}。
同时 {guardrail_metrics} 不应恶化。
```

示例占位：

- 用户在 `{scene}` 对 `{core_value}` 的需求最强，此处展示订阅权益会提升点击购买。
- `{country}` 对 weekly 价格敏感，增加 yearly 锚点可能提升年包占比和 LTV。
- 延长试用可能提升 trial start，但必须验证 trial conversion 和退款。

## 指标设计

| 类型 | 指标 | 说明 | 成功标准 |
| --- | --- | --- | --- |
| 主指标 | Paywall CVR / First Purchase Rate / Net Revenue / D7 LTV | 只能选 1 个主指标 | `{threshold}` |
| 收入指标 | ARPPU、ARPU、subscription revenue、net revenue | 要区分 gross 与 net | `{threshold}` |
| 漏斗指标 | exposure、purchase_click、store_sheet_show、trial_start、first_purchase | 定位损失点 | `{threshold}` |
| 长期指标 | trial conversion、renewal rate、D30 LTV、cohort ROI | 订阅必须看 | `{threshold}` |
| 护栏指标 | retention、session、refund rate、complaint、crash、unsubscribe | 任一恶化需暂停扩量 | `{threshold}` |

## 最小样本与观察窗口

| 产品类型 | 最短观察 | 不足时处理 |
| --- | --- | --- |
| 一次性 IAP | 至少 3 个完整自然日或覆盖核心流量周期 | 标为低置信度，不做全量 |
| Weekly 订阅 | 至少覆盖 trial start 与首购，最好观察到首次续订 | 未到续订日时只判断前端漏斗 |
| Monthly / Yearly 订阅 | 前期可看 trial 和首购，LTV 需延后复查 | 分阶段输出，不提前宣布长期成功 |
| 买量 cohort | 至少按 install cohort 看 D0/D7，D30 需要成熟后补 | 未成熟 cohort 标注 forecast 或 pending |

分层规则：

- 如果总体胜出但高价值国家失败，不可直接全量。
- 如果自然用户胜出但买量用户失败，需要结合 cohort ROI 判断是否会伤害投放。

- 如果某分层样本不足，只写趋势，不给强动作。

## 决策规则

| 结果 | 条件 | 决策 |
| --- | --- | --- |
| 上线 | 主指标达标，护栏稳定，关键国家和买量 cohort 不恶化 | 分阶段扩大，保留监控 |
| 继续观察 | 主指标方向正向但样本不足或订阅 cohort 未成熟 | 延长观察到指定日期 |
| 局部上线 | 总体一般，但特定国家/用户层稳定胜出 | 仅对胜出分层上线 |
| 停止 | 主指标无提升或护栏恶化 | 停止 treatment，记录原因 |
| 重新设计 | 漏斗断点清晰但假设未验证 | 修改 Paywall/价格/入口后重开实验 |

## Paywall 评审清单

| 模块 | 检查点 |
| --- | --- |
| 权益 | 是否清楚说明用户获得什么、多久获得、与免费版差异 |
| 价格 | 是否展示周期、总价、折扣、续订规则、本地货币 |
| Trial | 是否清楚说明试用期、转正时间、取消方式 |
| CTA | 是否唯一、明确、不会误导 |
| 信任 | 是否包含条款、隐私、恢复购买、取消说明 |
| 场景 | 是否出现在用户价值感最高的位置 |
| 付费用户保护 | 已订阅用户是否不再看到付费广告或重复 Paywall |
| 本地化 | 语言、币种、文化表达、价格锚点是否匹配国家 |

## 价格实验专项

| 价格动作 | 适用场景 | 必看指标 | 风险 |
| --- | --- | --- | --- |
| 提价 | 转化稳定、退款低、价值强 | CVR、net revenue、refund、renewal | 转化下滑、差评 |
| 降价 | 高意愿低支付、价格敏感国家 | first purchase、ARPPU、LTV | 低价用户质量差 |
| 年包锚点 | 年包权益强、长期使用明确 | yearly share、D30 LTV、renewal | 首购下降 |
| 增加试用 | 用户需要体验后付费 | trial start、trial conversion、refund | 低质量试用、转正低 |
| 取消试用 | 误用试用、转正低、退款高 | first purchase、net revenue、complaint | 入口转化下降 |
