# 归因与数据质量检查模板

## 使用场景

在任何预算、素材、国家放量、campaign 生命周期判断之前，先用本模板确认平台数据和 Snowball / BI 是否能对齐。若数据质量不是 `ready`，只能给数据修复建议或中低置信度观察建议，不能输出强预算动作。

## 必查字段

| 类型 | 字段 | 用途 | 缺失影响 |
| --- | --- | --- | --- |
| 平台花费 | spend、currency、timezone | 对账成本和日界线 | 不能判断 CPI / ROI |
| 平台安装 | installs、conversions、conversion_action | 对账 install 数量和事件口径 | 不能判断获客效率 |
| 命名映射 | account、campaign、adgroup/adset、country | 连接平台和 Snowball | 无法定位预算对象 |
| Snowball 回收 | cost、install、revenue、roi_0、roi_3、roi_7、profit | 决策主口径 | 不能给强预算动作 |
| 用户质量 | retention、关键事件、付费或广告行为 | 判断低价量是否真实有效 | 容易误判低 CPI |

## 差异判断规则

| 差异类型 | 可接受解释 | 风险判断 | 处理 |
| --- | --- | --- | --- |
| Spend 差异 | 时区、币种、平台延迟、退款或税费口径 | 差异持续或集中在单 campaign 时高风险 | 校验账户时区、币种和导入口径 |
| Install 差异 | SKAN / 隐私延迟、平台归因窗口、重复安装过滤 | 平台高但 BI 低时可能是低质或归因缺失 | 查 conversion action、MMP、campaign 命名 |
| 国家差异 | IP 国家、商店国家、平台定位国家不同 | 国家放量判断会失真 | 统一使用报告指定国家口径 |
| Campaign 缺失 | 新 campaign 命名未入库、UTM / campaign_id 缺失 | 无法迁移预算到具体对象 | 补映射后再判断 |
| Revenue 缺失 | IAA 延迟、IAP 订阅延迟、BI 未回传 | ROI 不能用于决策 | 标记 `partial` 或 `blocked` |

## 数据质量状态

| 状态 | 判定标准 | 允许建议 |
| --- | --- | --- |
| ready | 成本、install、国家、campaign、核心回收字段完整且差异可解释 | P0/P1/P2 预算和素材建议 |
| partial | 关键字段可用，但存在延迟、缺少子层级或部分国家缺失 | P1/P2，中低置信度 |
| blocked | 成本、install、回收或映射存在关键缺口 | 仅 `data_gap_check` |
| inactive_no_data | 观察窗无有效投放或无可用行 | `no_action_no_data` |

## 决策限制

- `ready` 才能给 P0/P1/P2 明确预算或素材动作。
- `partial` 只能给 P1/P2 或中低置信度建议。
- `blocked` 只能给数据修复建议。
- Meta / Google creative 级别若没有 Snowball ROI，只能写“基于 parent campaign/adgroup 回收质量推断”。
- 平台 ROAS、CTR、CPI 不能单独作为放量、暂停或迁移预算依据。
