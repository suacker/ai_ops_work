# 指标字典

## 使用规则

同名指标在所有报告中必须使用同一口径。若由于数据源限制必须使用不同口径，必须在报告中写明“临时口径”和适用范围。

## 买量指标

| 指标 | 定义 | 主要数据源 | 注意事项 |
| --- | --- | --- | --- |
| Spend | 投放花费 | Meta Ads / Google Ads / BI | 需注明平台口径还是 BI 口径 |
| Installs | 安装量 | 平台 / Snowball / BI | 平台安装和 BI 归因安装可能不同 |
| CPI | Spend / Installs | 平台 / BI | 样本小或归因异常时低置信度 |
| CTR | Clicks / Impressions | 平台 | 不能单独决定放量 |
| CPC | Spend / Clicks | 平台 | 需要结合 CTR 和后续回收 |
| CPM | Spend / Impressions * 1000 | 平台 | 受国家、版位、人群影响 |
| ROI0 | cohort 当日收入 / spend | Snowball / BI | 需注明是否 forecast |
| ROI7 | cohort 7 日收入 / spend | Snowball / BI | 预测值和真实值必须分开 |
| Profit | Revenue - Spend | Snowball / BI | 需注明收入窗口和成本范围 |

## 广告变现指标

| 指标 | 定义 | 主要数据源 | 注意事项 |
| --- | --- | --- | --- |
| Ads Revenue | 广告收入 | Snowball / AdMob / AppLovin | 需注明预估/最终收入 |
| ARPDAU | Ads Revenue / DAU | Snowball / BI | 受 DAU 质量和广告策略影响 |
| eCPM | Revenue / Impressions * 1000 | AdMob / AppLovin / Snowball | 不能单独解释收入涨跌 |
| IPU | Impressions / User | Snowball / BI | 反映展示强度，需结合留存和体验 |
| Fill Rate | Filled Requests / Requests | AdMob / AppLovin | 需从广告后台或聚合后台确认 |
| Show Rate | Impressions / Filled Requests | AdMob / AppLovin | 可反映展示链路问题 |
| Match Rate | Matched Requests / Requests | AdMob / AppLovin | 常用于排查填充异常 |

## IAP / 订阅指标

| 指标 | 定义 | 主要数据源 | 注意事项 |
| --- | --- | --- | --- |
| Paywall Exposure | 看到 Paywall 的用户或次数 | BI / App 事件 | 分母必须统一用户或次数 |
| Paywall CVR | 购买点击或购买 / Paywall 曝光 | BI / App Store Connect | 需统一分母 |
| Trial Start | 试用开始人数 | App Store Connect / BI | iOS 订阅重点指标 |
| Trial Conversion | 试用转付费比例 | App Store Connect / BI | 需等完整试用周期 |
| Renewal Rate | 续订人数 / 到期人数 | App Store Connect / BI | 按订阅周期拆分 |
| Refund Rate | 退款订单 / 订单 | App Store Connect / BI | 高退款会影响放量判断 |
| Subscription LTV | 订阅用户生命周期价值 | BI / App Store Connect | 需注明窗口和预测方式 |

## 产品与运营指标

| 指标 | 定义 | 主要数据源 | 注意事项 |
| --- | --- | --- | --- |
| D1 Retention | 安装后第 1 天回访比例 | Snowball / BI | cohort 口径 |
| D7 Retention | 安装后第 7 天回访比例 | Snowball / BI | 样本不足时低置信度 |
| DAU | 日活用户 | BI / Snowball | 需确认时区 |
| DNU | 新增用户 | BI / Snowball | 需确认安装或首开口径 |
| Core Action Rate | 完成核心行为用户占比 | BI / 事件系统 | 每个产品需单独定义核心行为 |
| Push Opt-out Rate | Push 退订或关闭比例 | BI / 推送后台 | 需结合频率和内容 |
| Crash-free Users | 无崩溃用户占比 | Crash 平台 | 影响留存和收入判断 |

## 格式规范

| 指标 | 展示格式 |
| --- | --- |
| ROI / ROAS / 留存 / 转化率 | 百分比，例如 `42.36%` |
| CPI / CPA | 美元，通常 2-4 位小数 |
| eCPM | 美元，2 位小数 |
| 花费 / 收入 / 利润 | 美元，2 位小数 |
| 样本不足 | 写 `样本不足` 或 `低置信度` |
| 缺失 | 写 `数据缺失` |
