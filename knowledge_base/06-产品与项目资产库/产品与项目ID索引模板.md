## 字段说明

| 字段 | 说明 | 维护要求 |
| --- | --- | --- |
| Project ID | Snowball/BI 项目 ID | 不得用产品简称替代 |
| 产品名 | 公司内部统一名称 | 必须与报告和 Wiki 标题一致 |
| 平台 | Android、iOS 或双端 | 双端产品必须能拆平台分析 |
| 包名 / Bundle ID | 应用唯一标识 | Android 和 iOS 分别维护 |
| 变现类型 | IAA、IAP、Subscription、Hybrid | 混合变现必须注明主要收入来源 |
| 主力国家 | 当前主要经营国家 | 用 ISO 国家或英文国家缩写，避免口语 |
| 阶段 | testing、active、scale、maintenance、paused | 阶段变化需要写原因 |
| Owner | 产品、BI 或业务负责人 | Owner 变更必须同步权限页 |

## 产品阶段定义

| 阶段 | 定义 | 允许动作 |
| --- | --- | --- |
| testing | 新产品或新市场验证中 | 小预算测试、埋点验证、商店页测试 |
| active | 日常经营产品 | 常规买量、变现优化、运营活动 |
| scale | 已证明回收和留存，进入放量 | 增预算、扩国家、扩素材 |
| maintenance | 收入稳定但增长有限 | 成本控制、版本维护、低频实验 |
| paused | 暂停买量或暂停经营 | 只保留监控和必要维护 |
