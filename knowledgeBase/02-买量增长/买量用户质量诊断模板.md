# 买量用户质量诊断模板

## 使用场景

用于判断买量带来的用户是否真实有效，避免只因为 CPI 低、CTR 高或平台 CPA 低就放量。适用于国家放量、素材测试、campaign 生命周期和预算迁移前的质量校验。

## 质量指标

| 类型  | 指标                                              | IAA 重点                 | IAP 重点        | 判断  |
| --- | ----------------------------------------------- | ---------------------- | ------------- | --- |
| 激活  | first_open、onboarding completion、关键权限完成         | 是否形成广告展示机会             | 是否进入核心价值路径    |     |
| 留存  | D1、D3、D7 retention                              | 直接影响广告展示和长期 ARPU       | 影响付费转化窗口      |     |
| 行为  | session、核心功能使用、内容消费、搜索/阅读/完成任务                  | 是否有足够 IPU 和 engagement | 是否体现付费意图      |     |
| 变现  | ads revenue、eCPM、IPU、IAP revenue、first purchase | 广告收入质量                 | 首购率、付费率、ARPPU |     |
| 回收  | ROI0、ROI3、ROI7、profit、payback                   | 回收速度和利润                | 付费延迟和 LTV     |     |

## 常见问题判断

| 现象 | 可能原因 | 验证方式 | 动作 |
| --- | --- | --- | --- |
| CPI 低但 ROI 弱 | 低质国家、人群误配、素材承诺过强 | 看留存、关键事件、收入拆分 | 不放量，查素材和国家 |
| CTR 高但留存弱 | 素材吸引点击但产品体验不匹配 | creative hook 与用户行为对比 | 重写 hook 和落地承诺 |
| 留存好但收入弱 | 变现入口弱、广告展示不足、付费链路弱 | 看 IPU、eCPM、首购率 | 交给产品/变现优化，预算谨慎 |
| ROI0 弱但 ROI7 改善 | 回收延迟或 IAP 延迟 | 看 cohort 曲线 | 延长观察，不急停 |
| 平台 ROAS 好但 BI ROI 弱 | 回传口径或 attribution 差异 | 做归因对账 | `data_gap_check` |
| 某国家量大质弱 | 国家 mix 或素材本地化问题 | 国家级指标和素材拆分 | 降预算或拆国家测试 |

## 用户质量分层

| 分层 | 判断标准 | 动作 |
| --- | --- | --- |
| 高质量 | 留存、关键事件、ROI / profit 均达标 | 放量候选 |
| 可验证 | 部分指标达标，样本或收入窗口不足 | 小预算观察 |
| 低质量 | CPI 低但留存、事件或收入弱 | 不放量，查原因 |
| 数据不足 | 样本低、收入延迟或映射缺失 | `too early to judge` / `data_gap_check` |
